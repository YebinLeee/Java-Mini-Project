# Java-Mini-Project

### 🖥️☑️ TodoList Java-Based Windows Application
<br>

## Application 기능
  - 새로운 업무(Task) 생성(index number 갱신)
  - 완료된 업무 색 변경
  - 완료된 업무 리스트에서 제거

<br>
